# Standard Deviation calculation

Usage

```javascript
const stDev = require('node-stdev');

stDev.sample([5, 5, 4, 2, 5]);			// OUTPUTS: 1.30

stDev.population([5, 5, 4, 2, 5]);		// OUTPUTS: 1.17
```
